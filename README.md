# Portfólio DS

Portfolio pessoal desenvolvido com Django, com uma homepage atualizada, visual técnico, navegação responsiva e melhorias de SEO para compartilhamento em redes sociais.

## Página Home atualizada

A home foi refinada para apresentar de forma direta:

- destaque para o nome e perfil profissional
- descrição principal com foco em programação, suporte e análise de sistemas
- dois botões de ação: "Ver Projetos" e "Entre em Contato"
- cards de funções com ícones visuais
- layout centralizado com identidade tecnológica em azul escuro e verde vibrante
- comportamento responsivo para dispositivos móveis

## Melhorias aplicadas

- navbar fixo com destaque visual e navegação clara
- menu mobile com botão hambúrguer
- refinamento do CSS para melhorar legibilidade, hierarquia e contraste
- meta tags Open Graph e Twitter Card
- favicon
- sitemap.xml e robots.txt

## Tecnologias

- Python
- Django 6.0.7
- SQLite
- HTML
- CSS
- JavaScript
- Font Awesome
- Google Fonts

## Como executar localmente

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
& .\portfolio-ds\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py runserver
```

Acesse:

```text
http://127.0.0.1:8000/
```

Projeto com homepage atualizada, layout responsivo, navegação mobile e SEO básico já implementados.
